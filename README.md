### 프로젝트 소개
Java와 Servlet/JSP를 학습한 후 만든 영화 예매 사이트

### :calendar: 기간
2024.04.08 ~ 2024.04.12  약 1주

### 주요기능
- 영화 검색
- 영화 예매/취소
- 영화 즐겨찾기/삭제
- 로그인/회원가입
- 회원 정보 수정

### 개발환경
- HTML5, CSS3, Javascript, jQuery
- Servlet/JSP
- Oracle 11.2.0.2.0
- Apache Tomcat-9.0.87
### TOOL 
- Eclipse IDE 4.30.0
- SQL Developer
